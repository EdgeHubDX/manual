# Jekyll 사용법 (임시)

## 1. Ruby 설치

- [설치 방법](https://jekyllrb-ko.github.io/docs/installation/windows/#jekyll-%EC%84%A4%EC%B9%98%ED%95%98%EA%B8%B0)

## 2. 프로젝트 클론

```bash
$ git clone [레포주소]
```

## 3. 플러그인 설치

```bash
$ bundle install
```

## 4. 서버 실행

```bash
$ bundle exec jekyll serve
```

## 5. 프로젝트 빌드

```bash
$ bundle exec jekyll build
```
